# 🖥️ System Info Viewer

## 🇧🇷 Descrição
Aplicação desenvolvida em Java com interface gráfica (Swing) para exibição de informações do sistema em tempo real.

O projeto permite visualizar dados como data, hora, idioma do sistema e resolução da tela, servindo como uma ferramenta simples de diagnóstico e aprendizado em Java GUI.

---

## 🇺🇸 Description
A Java application built with a graphical interface (Swing) to display real-time system information.

This project allows users to view data such as current date, time, system language, and screen resolution, serving as a simple diagnostic and learning tool for Java GUI.

---

## 🚀 Funcionalidades | Features
- Exibição da data atual  
- Exibição da hora atual  
- Identificação do idioma do sistema  
- Detecção da resolução da tela  
- Interface gráfica com botão de execução  

---

## 🧠 Tecnologias utilizadas | Tech Stack
- Java  
- Swing (GUI)  

---

## 📦 Pré-requisitos | Requirements
- Java JDK 8 ou superior  

---
